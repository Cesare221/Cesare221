## Sobre mim

Sou desenvolvedor Full Stack com foco em **desenvolvimento web**, **dashboards**, **automações**, **sistemas SaaS** e aplicações que exigem integração entre **interface, regras de negócio, API, banco de dados e deploy**.

Tenho interesse especial por projetos que resolvem problemas reais de operação, produtividade e organização de informação, seja em produtos de gestão, OCR, geração de documentos, sistemas internos ou ferramentas voltadas para fluxo de trabalho.

Hoje, meu foco está em construir soluções completas com boa base técnica, experiência clara para o usuário e estrutura preparada para evolução.
